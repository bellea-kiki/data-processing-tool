# 数据处理工具

一个现代化的数据处理和可视化工具，支持表格导入、数据展示和动态绘图。

## ✨ 核心功能

- **📁 数据导入**：支持 CSV 文件上传和手动数据输入
- **📋 表格显示**：清晰的数据表格展示，支持实时查看
- **📈 数据可视化**：
  - 支持折线图、柱状图、散点图多种图表类型
  - 灵活选择 X 轴和 Y 轴数据
  - 实时更新图表

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **图表库**：Chart.js + react-chartjs-2
- **样式**：CSS3 (带渐变和动画效果)

## 📊 使用指南

### 1. 导入数据
- **方式一**：点击"📤 上传 CSV 文件"上传本地 CSV 文件
- **方式二**：点击"✏️ 手动输入数据"直接输入 CSV 格式数据

CSV 格式示例：
\`\`\`
Date,Sales,Visits
2024-01-01,100,50
2024-01-02,150,75
2024-01-03,120,60
\`\`\`

### 2. 查看表格
导入后，数据将以表格形式展示，支持查看所有列和行。

### 3. 生成图表
- 选择**图表类型**（折线图/柱状图/散点图）
- 选择 **X 轴**数据列
- 选择 **Y 轴**数据列
- 图表实时更新
